# spritesheet-generator
create sprite sheets from layers &amp; layer groups

## Installation
[Download](https://github.com/Morlin1129/spritesheet-generator/raw/refs/heads/main/spriteshee-generator.jsx) the raw version of the script

Place Spritesheet generator.jsx script into:

Windows - C:\Program Files\Adobe\Your photoshop version\Presets\Scripts
Mac - Applications > Your photoshop version > Presets > Scripts
Restart Photoshop if it was open.

## Usage
The script will now be available under File -> Scripts menu as "Spritesheet Generator".

# スプライトシート生成スクリプト
スプライトシートを作成するPhotoshopのプラグインスクリプトです。

## 特徴
- レイヤーごとに作成したファイルを1つまたは複数のスプライトシートに出力します。
- すべてのレイヤーを1つのスプライトにするか、グループごとにスプライトを吐き出すか選択できます。
- スプライトは上から順番に生成されます。
